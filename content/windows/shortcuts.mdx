# ⌨️ Windows Keyboard Shortcuts

> Nützliche Tastenkombinationen für effizienteres Arbeiten unter Windows.

---

## 🚀 Allgemein

| Shortcut              | Funktion                            |
|-----------------------|-------------------------------------|
| `Win`                 | Startmenü öffnen                    |
| `Win + D`             | Desktop anzeigen/verstecken        |
| `Win + E`             | Explorer öffnen                    |
| `Win + L`             | Sperrt den Computer                |
| `Win + R`             | „Ausführen“-Dialog öffnen          |
| `Win + I`             | Einstellungen öffnen               |
| `Win + S`             | Suche öffnen                       |
| `Alt + Tab`           | Zwischen Fenstern wechseln         |
| `Ctrl + Shift + Esc` | Task-Manager direkt öffnen         |

---

## 🪟 Fensterverwaltung

| Shortcut              | Funktion                                 |
|-----------------------|------------------------------------------|
| `Win + ↑ / ↓`         | Fenster maximieren / minimieren          |
| `Win + ← / →`         | Fenster an linke/rechte Bildschirmhälfte |
| `Win + Shift + ← / →` | Fenster auf anderen Monitor verschieben  |
| `Win + Home`          | Alle anderen Fenster minimieren          |

---

## 📝 Texteingabe & Copy-Paste

| Shortcut            | Funktion                 |
|---------------------|--------------------------|
| `Ctrl + C / X / V`  | Kopieren / Ausschneiden / Einfügen |
| `Ctrl + Z / Y`      | Rückgängig / Wiederholen |
| `Ctrl + A`          | Alles markieren          |
| `Ctrl + F`          | Suchen                   |

---

## 🖼️ Screenshots

| Shortcut             | Funktion                     |
|----------------------|------------------------------|
| `Win + Shift + S`    | Bereichs-Screenshot öffnen   |
| `Print Screen`       | Ganzen Bildschirm kopieren   |
| `Alt + Print Screen` | Aktuelles Fenster kopieren   |

---

## 🔒 Sicherheit

| Shortcut     | Funktion                       |
|--------------|--------------------------------|
| `Ctrl + Alt + Del` | Sicherheitsmenü öffnen  |
| `Win + L`         | Benutzer sperren          |

---

> 💡 **Tipp**: Du kannst `Win + 1`, `Win + 2`, ... verwenden, um Programme in der Taskleiste direkt zu starten.
